## soal3

- Fungsi melakukan iterasi satu kali sepanjang string input sebanyak n karakter, sehingga kompleksitas waktu adalah O(n).

- Setiap karakter dibaca dan diproses satu kali, sehingga tidak ada nested loop yang menaikkan kompleksitas.

- Penggunaan stack untuk menyimpan bracket buka juga beroperasi dalam O(1) untuk push dan pop.

- Dalam kasus terburuk, seluruh karakter dalam string adalah bracket buka, sehingga stack dapat menyimpan hingga n elemen, sehingga kompleksitas memori adalah O(n).

- Kesimpulan: Fungsi ini sangat efisien untuk masalah pengecekan bracket seimbang dengan kompleksitas waktu linear dan ruang linear.
